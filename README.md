# Markdown Blog
This [markdown](https://www.computerhope.com/jargon/m/markdown.htm) blog is built with Node.js Express and MongoDB Atlas.

## Install
```
npm install express mongoose ejs
npm install --save-dev nodemon
npm install dotenv
```

## Deployment
```
npm run devStart
```
